Caso de Estudio: Alke Wallet

Descripción

Aplicación web dinámica para la gestión de usuarios y operaciones financieras básicas. Permite depósitos, transferencias y consulta de historial con persistencia real.

Desafío Principal

Evolucionar una estructura estática hacia una plataforma dinámica con integridad de datos y gestión de estados financieros en tiempo real.

Solución & Herramientas

Se utilizó el patrón **MVT de Django 6.0.3** e integración con **SQLite** mediante el **ORM**. La interfaz se construyó con Bootstrap 5.3 y JavaScript para validaciones visuales.

Impacto y Aprendizaje

  • Métricas: Cumplimiento del 100% de operaciones CRUD y persistencia íntegra de transacciones.
  • Aprendizaje: Dominio en migraciones, gestión de entornos virtuales (venv) y lógica de negocio en backend.

Justificación de elección: Este proyecto demuestra mi capacidad para iterar soluciones técnicas y adaptarlas a requerimientos reales del sector fintech, reflejando mi preparación como Trainee Fullstack.
